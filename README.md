# Unit 4 Lecture 2: 2D Arrays & Matrices

Interactive lecture material for C programming, focusing on the visualization and understanding of 2D arrays, memory mapping, and nested loops.

## Features
- **Row-Major Visualizer**: See how 2D arrays map to 1D physical memory.
- **Nested Loop Simulator**: Step-by-step execution of nested loops visiting matrix cells.
- **Address Calculator**: Interactive tool to verify the memory address formula.
- **Matrix Addition Lab**: hands-on matrix arithmetic.

## Getting Started

### Prerequisites
- Node.js installed

### Installation

```bash
npm install
```

### Local Development

```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) (runs at root).

### Deployment

This project is configured for GitHub Pages.
```bash
npm run deploy
```
*Note: Production builds serve from `/UNIT4L2` subpath.*
